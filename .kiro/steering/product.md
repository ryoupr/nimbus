# プロダクト概要

## EC2 Connect v3.0

高性能 EC2 SSM 接続管理ツール - Rust で実装された自動セッション管理機能付き CLI アプリケーション

## 主要機能

- **自動セッション維持**: セッションを自動的に監視し、切断を予防
- **高速再接続**: 5秒以内の切断検出と自動再接続（指数バックオフ対応）
- **セッション管理最適化**: 複数セッションの効率的な管理
- **パフォーマンス監視**: 接続速度とレイテンシの継続的な監視
- **リソース使用量最適化**: メモリ 10MB 以下、CPU 0.5%以下の軽量動作
- **リッチターミナル UI**: ratatui による美しいターミナルインターフェース
- **包括的診断機能**: 接続前チェック、自動修復、リアルタイム診断 UI
- **VS Code 統合**: SSH 設定の自動管理とシームレスな統合

## パフォーマンス目標

- メモリ使用量: 10MB 以下
- CPU 使用率: 0.5%以下（通常動作時）
- 接続時間: 150ms 以下
- 切断検出: 5秒以内

## ターゲットユーザー

- AWS EC2 インスタンスに頻繁に接続する開発者
- 複数の EC2 インスタンスを管理する DevOps エンジニア
- 安定した接続を必要とする本番環境の運用チーム
- CI/CD パイプラインで自動化を行うチーム
